# synyca

**synyca** is a minimal, modern **synonym and word meaning generator** built with pure HTML, Tailwind CSS, and vanilla JavaScript.  
It allows users to instantly discover **definitions, phonetics, examples, and synonyms** for any English word using public APIs.

---

## Features

- Search for any English word  
- Get clear **definitions**, **part of speech**, and **usage examples**  
- Displays **phonetic pronunciation**  
- Fetches up to **100 synonyms**  
- Click any synonym to instantly search it  
- Parallel API fetching for fast responses  
- Clean, minimal UI with smooth animations  
- Fully responsive (desktop & mobile)
